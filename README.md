# Spotify Data Analysis using Python

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a Spotify dataset using Python to uncover trends in music popularity, audio features, genres, and song characteristics. The analysis includes data cleaning, visualization, and statistical exploration to derive meaningful insights from the dataset.

---

## Objectives
- Clean and preprocess Spotify dataset.
- Explore relationships between audio features and song popularity.
- Analyze trends across genres and release years.
- Visualize distributions and correlations using Python.
- Extract actionable insights through exploratory data analysis.

---

## Dataset
The dataset contains information about Spotify tracks, including:
- Track Name
- Artist
- Genre
- Popularity
- Danceability
- Energy
- Loudness
- Acousticness
- Valence
- Tempo
- Duration
- Release Year

## Data Cleaning
The following preprocessing steps were performed:
- Handled missing values.
- Removed duplicate records.
- Checked data types.
- Performed basic feature selection.
- Prepared the dataset for visualization and analysis.

---

## Exploratory Data Analysis
The analysis includes:
- Correlation analysis between audio features.
- Popularity distribution of songs.
- Genre-wise comparisons.
- Relationship between loudness and energy.
- Song duration analysis.
- Year-wise music trends.
- Feature distribution using statistical visualizations.

---

## Key Insights
- Strong positive correlation observed between energy and loudness.
- Acoustic tracks generally showed lower popularity.
- Music characteristics varied across different genres.
- Song popularity is influenced by multiple audio features rather than a single factor.
- Data visualization helped identify trends that are difficult to observe from raw data alone.
